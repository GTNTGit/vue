# Crypto Exchange Mobile Frontend

这是一个模仿 XT.com, OKX, Binance 的移动端加密货币交易所前端项目，使用 Vue 3 + Vite 开发。

## 🛠️ 环境准备 (Prerequisites)

在运行项目之前，你需要安装 Node.js。

1.  **下载 Node.js**: 访问 [Node.js 官网](https://nodejs.org/) 下载并安装最新版本 (LTS 版本即可)。
2.  **验证安装**: 打开终端 (Terminal)，输入 `node -v` 和 `npm -v`，如果有版本号输出说明安装成功。

## 🚀 快速开始 (Quick Start)

1.  **安装依赖**:
    在项目根目录下运行：
    ```bash
    npm install
    ```

2.  **启动开发服务器**:
    ```bash
    npm run dev
    ```

3.  **预览**:
    启动成功后，终端会显示一个链接（通常是 `http://localhost:5173`）。在浏览器中打开该链接即可实时预览界面。

## 📱 移动端预览

如果想要在手机上预览：
1. 确保手机和电脑连接在同一个 Wi-Fi 网络。
2. 运行 `npm run dev -- --host`。
3. 终端会显示一个 Network 链接（例如 `http://192.168.x.x:5173`），在手机浏览器中输入该地址即可。
